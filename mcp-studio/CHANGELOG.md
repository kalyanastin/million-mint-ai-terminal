# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-27

### Added
- Initial monorepo layout setup (`pnpm`, `typescript`, `eslint`, `prettier`).
- Node-based MCP Runner daemon for browser proxy execution.
- Unified JS/TS client API process spawner core contract.
- React-based dashboard UI featuring log tracers, stdout timeline, and connection statuses.
- Automated GitHub Actions build and release workflows.
