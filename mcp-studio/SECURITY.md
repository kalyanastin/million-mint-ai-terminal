# Security Policy

## Supported Versions

Only the latest major release of MCP Studio is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of MCP Studio seriously. If you find any vulnerability, please do not report it as a public issue. Instead, report it privately.

Please report security issues by emailing:
**security@mcpstudio.dev**

Once we receive your report, we will:
1. Acknowledge receipt within 48 hours.
2. Investigate and verify the vulnerability.
3. Coordinate a patch and release a security advisory alongside a minor/patch update.
