# MCP Studio Ecosystem Roadmap

This document outlines the high-level goals and planned releases for the MCP Studio project ecosystem.

## Planned Releases

### v0.1.0 — MVP Launch (Current Focus)
* [ ] Working monorepo structure with pnpm
* [ ] Core React GUI app with modern Obsidian dark theme
* [ ] WebSocket process runner daemon to spawn local commands
* [ ] Stdio process stdin/stdout communication logs
* [ ] Live JSON-RPC request/response debugger timeline

### v0.2.0 — Interactive Forms & Inspection
* [ ] Dynamic form engine based on Zod and JSON-Schema (`@mcp-studio/schema-form`)
* [ ] Hierarchical search-enabled JSON viewer (`@mcp-studio/json-inspector`)
* [ ] Workspace settings and config profiles storage
* [ ] WebSocket/SSE remote MCP server support

### v1.0.0 — Ecosystem Release
* [ ] Production ready native desktop app wrapper via Tauri v2
* [ ] Floating devtools overlay widget (`@mcp-studio/devtools`)
* [ ] TypeScript integration hooks (`@mcp-studio/react`)
* [ ] Community server registry showcase integrations
