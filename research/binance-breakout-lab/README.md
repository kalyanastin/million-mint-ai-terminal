# Million Mint Binance Breakout Strategy Research Lab

This is an isolated research laboratory environment built to study, backtest, and evaluate open-source Binance trading bots and breakout systems.

## Directory Structure

```text
research/
└── binance-breakout-lab/
    ├── README.md                 # Lab documentation (this file)
    ├── GITHUB_BREAKOUT_RESEARCH.md # GitHub research audit of candidate bots
    ├── BACKTEST_FRAMEWORK.md     # Quantitative backtesting methodology guide
    ├── PERFORMANCE_DASHBOARD.md  # Dynamic performance dashboard results
    ├── million_mint_framework.py # Pure Python trading & simulation framework
    ├── requirements.txt          # Python environments guidelines
    ├── sample_data/              # Data storage
    │   └── README.md
    ├── notebooks/                # Jupyter notebook scratchpads
    │   └── research_scratchpad.ipynb
    └── outputs/                  # Backtest metrics outputs
        ├── backtests/
        ├── reports/
        └── charts/
```

## Running the Simulator

To run the simulation and regenerate the performance metrics and dashboard, execute the following command from the workspace root:

```bash
python research/binance-breakout-lab/million_mint_framework.py
```
